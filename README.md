# Evolve X 🌱

Evolve X is a Django-based Carbon Footprint Tracker that uses Machine Learning to analyze and estimate carbon emissions based on user activities. The platform also provides personalized carbon reduction recommendations to help users adopt more sustainable practices.

🛠️ Tech Stack
- Python
- Django
- Pandas
- NumPy
- HTML
- CSS
- SQLite


📌 Features
- Carbon footprint calculation and tracking
- Machine Learning-based emission analysis
- Personalized carbon reduction recommendations
- User-friendly dashboard
- Activity-based emission monitoring
- Sustainable lifestyle insights
- Responsive web interface




▶️ Run the Django Server
- python manage.py runserver

The application will start locally at:

http://127.0.0.1:8000

🌍 Carbon Reduction Recommendations:

-The system provides smart recommendations such as:

- Reducing energy consumption
- Using public transportation
- Minimizing fuel usage
- Sustainable lifestyle suggestions
- Improving energy efficiency
