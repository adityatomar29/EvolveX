# Evolve X 🌱

A Full stack Carbon Footprint Tracker built with Django that helps users measure, analyze, and reduce their environmental impact. Evolve X calculates carbon emissions based on daily activities and provides personalized recommendations to promote sustainable living and environmentally conscious decision-making.

## 🚀 Live Demo

🔗 **Deployed Application:**
[Visit Evolve X](https://adit29.pythonanywhere.com)

---

## 📖 Project Overview

Evolve X is designed to raise environmental awareness by enabling users to track their carbon footprint and understand how their lifestyle choices contribute to carbon emissions. Using user defined techniques, the platform analyzes activity data and generates actionable recommendations to help users reduce their environmental impact.

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/adityatomar29/EvolveX.git
cd evolve-x
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

### 3. Activate the Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/macOS

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Apply Database Migrations

```bash
python manage.py migrate
```

### 6. Create a Superuser (Optional)

```bash
python manage.py createsuperuser
```

### 7. Run the Development Server

```bash
python manage.py runserver
```

The application will be available at:

```text
http://127.0.0.1:8000/
```
